# Balaji eMitra Premium Website

Premium static multi-page website for **Shree Nohariya Ballaji Emitra and CSC Centre**

## Pages
- index.html
- services.html
- forms.html
- pricing.html
- about.html
- faq.html
- contact.html
- 404.html

## Folder Structure
- assets/css/style.css
- assets/js/script.js
- assets/js/forms-data.js
- assets/images/

## Features
- Premium color theme
- Mobile responsive layout
- Sticky navbar
- Icons and card-based design
- Hero animation
- FAQ accordion
- Dynamic forms search and filter
- GitHub Pages compatible

## Edit Guide
- Change colors in `assets/css/style.css`
- Edit forms in `assets/js/forms-data.js`
- Update contact info in all HTML pages
- Replace `#` links with real form or PDF links

## GitHub Upload Steps
1. Create a new GitHub repository
2. Upload all files
3. Commit changes
4. Open **Settings > Pages**
5. Select **Deploy from a branch**
6. Choose **main** and **/(root)**
7. Save

## Custom Domain
For custom domain like `balajiemitra.in`, add a `CNAME` file in root with your domain name and connect DNS from your domain provider.
